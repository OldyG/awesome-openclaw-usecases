# Contributing

## Adding a Use Case

1. `/usecases` 폴더에 사용 사례 이름으로 마크다운 파일을 만드세요.
2. 사용 사례를 자세히 추가하세요. 다루면 좋은 항목은 다음과 같습니다: 
   - **Pain Point** — 이 문제가 무엇을 해결하나요?
   - **What It Does** — 사용 사례에 대한 간단한 설명
   - **Prompts** — OpenClaw에 제공할 실제 프롬프트/설정
   - **Skills Needed** — 어떤 OpenClaw 스킬이 필요한지
   - **Related Links** — 관련 문서, API, 도구
3. 메인 [README.ko.md](README.ko.md)의 해당 카테고리 테이블에 행을 추가하세요.
4. PR을 오픈하세요.

## Guidelines

- 마크다운당 하나의 사용 사례
- 설명은 간결하지만 복제할 수 있을 정도로 충분히 작성하세요
- 사용 사례가 기존 카테고리에 맞지 않으면 PR에서 새 카테고리를 제안하세요
- 접근 방식이 의미있게 다르면 중복도 허용됩니다
- 새로운 사용 사례를 AI로 생성하지 마세요. 실제로 테스트하고 검증한 사용 사례만 제안하세요.
- **No crypto-related use cases** — 이는 수락되지 않습니다.
